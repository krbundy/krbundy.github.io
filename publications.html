Wireless Sensing of Forest Ecosystem Using Artificial Intelligence and Machine Learning

Sonia Naderi, Kenneth Bundy, Thayer Whitney, Victoria Nicholas, Ali Abedi, Aaron Weiskittel, Alexandra Contosta

Description: We investigate a design for low-cost wireless sensor systems for widespread forest monitoring. We use soil moisture sensors and low-power ARM Cortex-M3 based microcontrollers to create the backbone for the system. To this, we add a variety of machine learning-based enhancements, specifically packet scheduling. We are also investigating methods for machine learning based accuracy enhancements, routing, and other aspects of the wireless sensor network that can be improved with machine learning on embedded devices.
